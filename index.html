Chronos

Automate your event capturing

Chronos provides an intuitive configuration file that lets you specify which events you'd like to capture. Afer that, just drop the tracker in your webpage and you're all set - that's it.

Learn More

Built to Stream from the Ground Up

Leverage the power of Apache Kafka to stream your event data. Add new producers and consumers at will - everything is decoupled.

Databases Built for Events

Chronos utilizes time-series databases to efficiently store your data for analysis

Use TimescaleDB to explore your data and discover usage trends

Use PipelineDB to aggregate your data into realtime results
